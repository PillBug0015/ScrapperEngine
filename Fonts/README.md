# Game Fonts Directory

Place your custom TrueType Font (`.ttf` or `.otf`) in this directory and rename it to **`default.ttf`**.

The **Scrapper Engine** will automatically detect it on startup, load it with full Hangul (Korean) unicode glyph support, and use it to render all in-game text!

### Recommended Free Korean Fonts:
- **Nanum Gothic** (나눔고딕)
- **Nanum Square** (나눔스퀘어)
- **D2Coding** (for coding/retro style)
- **MapleStory Font** (메이플스토리체)
